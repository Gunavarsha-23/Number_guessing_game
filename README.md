# Number Guessing Python Game

A classic terminal-based Number Guessing game built using Python. Put your instincts to the test and see how fewer attempts you can take to find the secret number!

## 🚀 Features
* **Multiple Difficulties:** Choose between **Easy mode** (10 attempts) and **Hard mode** (5 attempts) depending on your confidence.
* **Smart Feedback:** The game guides you after every guess, telling you if your number is "Too high" or "Too low".
* **Attempt Tracker:** Keeps a live count of your remaining guesses so you can plan your next move.
